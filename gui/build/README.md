# Build Resources

Place your application icons here:

## Icon Requirements

- **Windows**: `icon.png` or `icon.ico` (256x256 minimum)
- **macOS**: `icon.png` or `icon.icns` (1024x1024 recommended)
- **Linux**: `icon.png` (512x512 minimum recommended)

Place your icon files in this directory and they will be used during the build process.

For development, you can generate icons from a high-resolution PNG using:
- Windows: Use an online converter or tools like `electron-icon-builder`
- macOS: Use online converters or `iconutil` on macOS
- Linux: `.png` file works directly
